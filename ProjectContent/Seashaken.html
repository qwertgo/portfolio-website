Overview

An atmospheric 3D strategy game casting players as a lighthouse keeper navigating ships through dangerous waters, delivering fish to surrounding islands, and uncovering mysterious creatures lurking in dark depths.

Implemented Features

Ship Navigation System

  • Developed a comprehensive route planning system supporting diverse mission types including fishing and cargo delivery
  • Enabled route modification through adding, deleting and inserting tasks
  • Implemented object interaction allowing ships to respond to fishing grounds, islands, and game system triggers
  • Designed internal path smoothing algorithms to produce natural, believable ship movement
  • Integrated visual feedback for route visualization and ship status indication

Enemies and Obstacles

  • Built three distinct enemy AI variants with unique behavioral patterns capable of interfering with ship navigation or ship inventory
  • Developed a probability based spawning system that adjusts enemy and obstacle placement to ensure balanced density while preventing overlap
  • Created a tagging system that allows users to mark and track discovered enemies and obstacles throughout the level

Visual Contributions

  • Authored a custom water shader featuring a paper art aesthetic with mapline overlays, stepped depth mapping (driven by terrain for artist friendly adjustments), and multi-directional current flows without visible blending artifacts.
  • Designed a conditional visibility shader where enemies remain hidden outside the lighthouse spotlight but appear red when marked by the player.
  • Shaped Fog of War to fit into the map aesthetic, ensuring it harmonizes with the stylized water and environment
  • Built an adjustable dotted line shader entirely for flexible path and boundary visualization.
  • All Shaders created using Unity Shader Graph sometimes accelerating them with custom hlsl code
